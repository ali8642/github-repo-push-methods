How to Create and Host a Github Repository

This guide will help you create a new repositories in Github, upload projects and host it using Github pages.

1. Creating a new repository

  1. Log into your Github account
  2. Click on the New button or the + icon and select "New repository".
  3. Enter a repository name and description.
  4. Choose public or private visibility.
  5. Click on Create repository button.

2. Uploading your project

  • Using Git (Command Line):
    1. Download Git from Google and install it first.
                git init
            git add .
            git commit -m "first commit"
            git branch -M main
            git remote add origin https://github.com/yourusername/yourrepo.git
            git push -u origin main

            
    
            After this, sign in screen of Github will appear if you are pushing your repository in the Github for the very first time using Git Command.

            You can also use CMD of your windows by using the same git commands.
             
            Note:
            You can add any comment or description in the place of "first commit".
            You have to add your git repository link instead of the example link in the fifth line.
  • Using Github Web Interface:

    • Go to your repository, click "Add file" -> "Upload files", then drag and drop your project files, and click on "Commit changes".

  • Using Github Desktop
    1. Download Github Desktop from Google, install it and then sign in using your Github Account.
    2. Choose the option "Create repository"
    3. enter the repo name. Use "-" instead of " " (space).
    4. Publish your repo and choose tick the checkbox if you want to keep your repo private.

3. Hosting with Github Pages

Method Steps
From Repository Settings Go to "Settings" -> Select branch to main -> "Save".
Custom Domain Set your custom domain under "Custom domain" in Github Pages settings.
